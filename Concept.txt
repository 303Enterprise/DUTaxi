Концепция минимального функционала приложения ДУшноеТакси

1. Зарегестрироваться:
	- ввести свою ссылку на профиль ВК
	- ввести свой номер телефона
	- придумать пароль
1.1. Примечания:
	- возможно ли сделать запоминание пользователя? Чтобы не вводить пароль каждый раз
	- можно не включать в минимальный функцонал, но придется каждый раз вводить данные

2. Оставить запрос на поездку:
	- указать время сбора для поездки
	- указать место сбора для поездки (Двойка / УНИКС / ДУ / КПП / Кастомный адрес)
	- указать точку прибытия
	- указать количество оставшихся свободных мест
	- опционально: указать примерную стоимость поездки

3. Просмотреть пул запросов на поездку:
	- увидеть время сбора для поездки
	- увидеть место отправления
	- увидеть место прибытия
	- увидеть количество свободных мест, в реальном времени
	- увидеть примерную стоимость поездки 
3.1. Примечания:
	- если дать возможность пользователю указывать стоимость поездки могут быть проблемы,
	  лучше сделать через API сравнение и расчет наименьшей стоимости, с указанием компании, 
	  где лучше сделать заказ для оставившего реквест
	- если использовать API, то можно с определенной периодичностью пересчитывать стоимость поездки
	  и отображать ее в реальном времени

4. Просмотреть конкретный запрос на поездку:
	- увидеть время сбора для поездки
	- увидеть место отправления
	- опцинально: получить возможность увидеть место отправления на карте
	- увидеть место прибытия
	- опцинально: получить возможность увидеть место прибытия на карте
	- увидеть количество свободных мест, в реальном времени
	- увидеть примерную стоимость поездки 
	- получить возможность ответить на запрос

5. Ответить на конкретный запрос на поездку:
	- получить ссылку на ВК организатора поездки
	- получить номер телефона организатора поездки
	- увидеть таймер, показывающий время до сбора
	- получить остальной функционал просмотра запроса на поездку

6. Смотреть статус своего запроса на поездку: 
	- увидеть сколько осталось времени до сбора
	- уменьшить количество оставшихся свободных мест
	- увидеть ссылки на ВК всех, кто ответил на запрос
6.1. Примечания:
	- одновременно можно иметь только один запрос на поездку
